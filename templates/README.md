# 公司名单模板说明

`company_list_template.xlsx` 是 Choice 公告下载助手的批量导入模板，用于填写需要在 Choice 金融终端中逐个搜索和下载公告的公司名单。

## 模板结构

模板包含两个工作表：

| 工作表 | 用途 |
| --- | --- |
| `Companies` | 公司名单填写区，程序读取这里的 `公司名称` 列。 |
| `说明` | 模板填写规则和注意事项。 |

## Companies 工作表

默认列如下：

| 列名 | 是否必填 | 程序是否读取 | 说明 |
| --- | --- | --- | --- |
| `公司名称` | 是 | 是 | Choice 中用于搜索公司的名称，请保持列名完全一致。 |
| `代码或简称（可选）` | 否 | 否 | 仅供人工记录，例如股票代码、简称或内部备注。 |
| `备注（可选）` | 否 | 否 | 仅供人工记录，可填写处理说明。 |

## 填写规则

- 第一行必须保留列名 `公司名称`。
- 公司名称从第二行开始填写。
- `公司名称` 列不能为空。
- 示例行可以删除或替换为真实公司名称。
- 其他列可以保留、修改或删除，程序只读取 `公司名称` 列。
- 建议保存为 `.xlsx` 或 `.xlsm` 格式。

## 示例

| 公司名称 | 代码或简称（可选） | 备注（可选） |
| --- | --- | --- |
| Microsoft Corp | MSFT | 示例：可删除 |
| Apple Inc | AAPL | 示例：可删除 |
| Tesla Inc | TSLA | 示例：可删除 |

## 使用方式

1. 打开 `company_list_template.xlsx`。
2. 在 `Companies` 工作表中填写公司名称。
3. 删除不需要的示例行。
4. 保存文件。
5. 在软件界面点击“Excel 文件”旁的“选择文件”，选择保存后的模板文件。
6. 软件会自动读取 `公司名称` 列，并显示已加载的企业数量。

## 常见问题

| 情况 | 处理方式 |
| --- | --- |
| 软件提示未找到“公司名称”列 | 检查第一行列名是否完全等于 `公司名称`，不要添加空格或改成其他名称。 |
| 软件显示 Excel 文件为空 | 确认第二行开始已经填写公司名称。 |
| 只想运行部分公司 | 在软件界面设置“提取范围”，无需拆分 Excel。 |
| 想保留股票代码或备注 | 可以填写在可选列中，程序会忽略，不影响运行。 |

真实业务公司名单通常包含敏感信息，发布 GitHub 或共享项目时请不要把填写后的 Excel 文件放入仓库。
